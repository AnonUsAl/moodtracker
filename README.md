# MoodTracker(beta)

个人情绪追踪工具。通过命令行快速记录每天的心情和状态，查看历史记录和统计趋势。

## 为什么

持续记录情绪波动有助于识别规律，对自己和医生都有参考价值。这个工具把数据存在本地 JSON 文件里，不依赖任何第三方服务，隐私完全自己掌控。

## 安装

不需要安装任何依赖，只需要 Python 3。

```bash
git clone https://github.com/AnonUsAl/moodtracker.git
cd moodtracker
```

## 使用

### 记录心情

```bash
python moodtracker.py log
```

会提示你输入：

- 情绪分数（1-5）
- 备注（可选，直接回车跳过）

### 查看历史

```bash
python moodtracker.py history
```

显示最近 7 条记录：

```
  2026-07-14 17:57  ████░  4/5  今天开始了一个新项目，感觉不错
  2026-07-14 18:30  ██░░░  2/5  有点疲惫
```

### 查看统计

```bash
python moodtracker.py stats
```

显示总记录数、平均分、最高/最低、分布图：

```
  总记录数:  3
  平均情绪:  3.0/5
  最高:      4/5
  最低:      2/5

  分布:
    5/5   (0)
    4/5  █ (1)
    3/5  █ (1)
    2/5  █ (1)
    1/5   (0)
```

## 数据

所有记录存储在 `data/records.json`，格式如下：

```json
[
  {
    "date": "2026-07-14",
    "time": "17:57",
    "mood": 4,
    "note": "今天开始了一个新项目，感觉不错"
  }
]
```

数据完全在本地，不上传任何地方。

## 情绪分数参考

| 分数 | 含义 |
|------|------|
| 1 | 很低 |
| 2 | 低 |
| 3 | 一般 |
| 4 | 好 |
| 5 | 很好 |
